📌 Overview :

This repository contains a collection of TypeScript programs developed while learning and practicing modern JavaScript with static typing. It demonstrates core programming concepts along with TypeScript-specific features such as types, interfaces, optional parameters, rest parameters, and object-oriented programming.

The repository focuses on writing type-safe, scalable, and maintainable code using TypeScript.

-------------------------------------------------------------------------------------------------------------------------------------------------------------------

🎯 Objectives :
Build a strong foundation in TypeScript and modern JavaScript (ES6+)
Understand static typing and type safety
Practice functional and object-oriented programming
Improve code structure and readability

-------------------------------------------------------------------------------------------------------------------------------------------------------------------

🧠 Topics Covered :

🔹 Basics
Data types (Datatype.ts, typeof.ts)
Variables and basic syntax (First.ts)
🔹 Control Flow
Conditional and looping constructs
For.ts, While.ts, Loop.ts, Loop1.ts, Loop2.ts
🔹 Functions
Function definitions and usage
Function.ts, FunctionDemo.ts, FunctionMax.ts
Named & anonymous functions
NamedFunction.ts, Anonymous.ts
Default, optional, and rest parameters
Default.ts, Optional.ts, Optional2.ts, Rest.ts, Rest2.ts
🔹 Arrays & Operations
Array handling and operations
Array.ts, Array2.ts, ArrayFunction.ts
🔹 Functional Programming
Lambda expressions & statements
LambdaExpression.ts, LambdaStatement.ts
🔹 Object-Oriented Programming (OOP)
Class-based implementations
OOP1.ts → OOP5.ts
Demonstrates encapsulation and modular design
🔹 Programming Paradigms
Procedural Programming (POP.ts)
Combination of structured and modular coding
🔹 Miscellaneous
Practice and test files (Test18.ts, Addition.ts, etc.)

-------------------------------------------------------------------------------------------------------------------------------------------------------------------

⚙️ How to Run :
🛠️ Prerequisites
Node.js installed

TypeScript installed globally:

npm install -g typescript
▶️ Steps

Clone the repository:

git clone https://github.com/your-username/TypeScript-Programs.git

Navigate to the folder:

cd TypeScript-Programs

Compile TypeScript to JavaScript:

tsc filename.ts

Run the compiled file:

node filename.js
